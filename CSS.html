CSS 学习

背景

CSS 背景属性用于设置元素的背景样式。常用的背景属性包括：

  • background-color: 设置背景颜色。
  • background-image: 设置背景图片。
  • background-repeat: 定义背景图片是否重复。
  • background-position: 定义背景图片的位置。
  • background-size: 定义背景图片的尺寸。
  • background-attachment: 定义背景图片是否固定在视口或随内容滚动。

文本

CSS 文本属性用于设置文本的样式和布局。常用的文本属性包括：
文本的对齐方式：
text-align: 定义文本的水平对齐方式（left、right、center、justify）。
text-indent: 定义文本的首行缩进。
line-height: 定义文本行间距。
letter-spacing: 定义字符之间的间距。
word-spacing: 定义单词之间的间距。
text-decoration: 定义文本装饰（underline、overline、line-through）。
text-transform: 定义文本转换（uppercase、lowercase、capitalize）。
text-shadow: 定义文本阴影效果。
font-size: 18px : 定义文本大小。
font-family: 定义文本字体。
color: 定义文本颜色。
